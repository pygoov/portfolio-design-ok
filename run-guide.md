# Как запустить проект

Это сайт на `Eleventy`.

Для работы нужно:
- `Node.js` — чтобы запускать проект
- `npm` — чтобы устанавливать зависимости и запускать команды

## Первый запуск

Открой терминал в папке проекта `portfolio-design-ok` и выполни:

```powershell
& "C:\Program Files\nodejs\npm.cmd" install
```

Это установит все зависимости проекта.

## Локальный просмотр сайта

Чтобы запустить сайт локально, выполни:

```powershell
& "C:\Program Files\nodejs\npm.cmd" run dev
```

Это нужно, чтобы:
- открыть сайт у себя на компьютере
- видеть изменения почти сразу после правок

## Сборка сайта

Чтобы собрать готовую версию сайта, выполни:

```powershell
& "C:\Program Files\nodejs\npm.cmd" run build
```

Это нужно, чтобы:
- проверить, что сайт собирается без ошибок
- обновить папку `docs/` с готовым результатом

## Что важно помнить

- Исходники сайта лежат в `site-src/`
- Контент лежит в `content/`
- Картинки для сайта лежат в `media/web/`
- Папку `docs/` вручную обычно не редактируют

## Самая короткая версия

```powershell
& "C:\Program Files\nodejs\npm.cmd" run dev
```

Это команда для обычной ежедневной работы над сайтом.
